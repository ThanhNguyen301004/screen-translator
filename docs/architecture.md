# System Architecture

## Overview

The system is modular, with components for UI, capture, OCR, translation, and overlay.

## Components

- **UI**: Handles user interaction.
- **Capture**: Screenshoting using mss.
- **OCR**: Text extraction with Tesseract.
- **Translation**: Using googletrans.
- **Overlay**: Display with Tkinter.
